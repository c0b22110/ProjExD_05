# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターのダニエルをキーボードを使って操作するゲームで、アイテム等を使い敵を倒したりしBESTSCOREを目指すゲーム。

## ゲームの実装
###共通基本機能
* 背景画面スクロールに関するクラス
* 主人公に関するクラス
### 担当追加機能
* 主人公の操作(jump)    (担当：平岡)
* score関係+コイン  (担当：高橋)
* 障害物(unnko)+GAMEOVER表示  (担当：島多)
* scoreに応じて出世（パワーアップ　(担当：中條)
* 無敵モードの実装  (担当：永田)
